## 🎮 Your First Task

After starting the app:

1. Click **"Register"** and create account
2. Go to **"Tasks"** tab
3. Click **"+ Create Task"**
4. Fill in:
   - Title: "My First Task"
   - Priority: "High"
   - Category: "Personal"
   - Estimated Time: 5 min
5. Click **"Create Task"**
6. Mark it complete ✓
7. Check Dashboard for XP! 🎉

## 🔐 Important Security Notes

⚠️ Before production deployment:

- [ ] Change JWT_SECRET in .env
- [ ] Update CORS_ORIGIN for your domain
- [ ] Use MongoDB Atlas (not local)
- [ ] Enable HTTPS
- [ ] Set NODE_ENV=production
